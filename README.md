# 🚀 Dự án Cuối kỳ MI4344: Vi xử lý MIPS 32-bit Pipelined (5 Tầng)

## 📁 PHẦN 1: CẤU TRÚC THƯ MỤC CỦA NHÓM

Mọi người phải lưu file đúng chỗ, tuyệt đối không vứt file lung tung ra ngoài.

* `asm/`: Chứa file code Assembly test (`.s`) và file mã máy (`.hex`) xuất ra từ MARS.
* `src/`: **LÀM VIỆC CHÍNH Ở ĐÂY**. Chứa code phần cứng (Verilog `.v`), ví dụ: `alu.v`, `regfile.v`...
* `tb/`: Chứa code Testbench (Môi trường test để cấp điện và xung nhịp cho chip hoạt động).
* `waves/`: Chứa các file cấu hình màn hình hiển thị sóng của GTKWave.
* `README.md`: Chính là file bạn đang đọc.
* `CONTEXT.md`: File quy ước kiến trúc kỹ thuật dành cho AI Agents.
